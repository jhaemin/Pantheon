# Radix Studio

## Development

### Runtime

- [Bun](https://bun.sh) >= 1

### Install dependencies

```zsh
bun install
```

### Run development server

```zsh
bun run dev
```

### Test

```zsh
bun test
```

### Build

```zsh
bun run build
```
